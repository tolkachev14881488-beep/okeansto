# Админка сайта СТО ОКЕАН

## Вход

После публикации на сайте откройте:

**https://okeancto.by/admin/**

Пароль по умолчанию: `okean2026` — смените в файле `admin/config.json` (не публикуйте свой пароль в открытый чат).

## Как сохранить изменения

1. Отредактируйте тексты во вкладках.
2. Нажмите **«Сохранить JSON»** — скачается `content.json`.
3. На GitHub в репозитории `okeansto` замените файл **`data/content.json`** на скачанный.
4. Подождите 2–3 минуты и обновите сайт (Ctrl+F5).

## Файлы

| Файл | Назначение |
|------|------------|
| `data/content.json` | Все тексты и телефоны для сайта |
| `js/site-content.js` | Подставляет JSON на страницу |
| `admin/index.html` | Панель редактирования |
| `admin/config.json` | Пароль (в .gitignore — создайте локально) |

## Безопасность

- Админка **не подключается к GitHub автоматически** — только скачивание файла.
- Пароль хранится в `admin/config.json` — это простая защита от случайных гостей, не от взлома.
- Для серьёзной защиты позже можно подключить вход через GitHub (Decap CMS).

## Локальная проверка

Откройте папку проекта и запустите любой локальный сервер, например:

```bash
cd Projects/sto-ocean
python -m http.server 8080
```

Сайт: http://localhost:8080/  
Админка: http://localhost:8080/admin/
